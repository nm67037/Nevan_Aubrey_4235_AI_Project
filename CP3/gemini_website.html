The Project

What is the RP4 Project?

The RP4 Motor Project is a high-precision, web-controlled motor system built around the versatile Raspberry Pi 4. It combines the power of Python scripting with robust hardware to deliver precise control over a stepper motor from any device on the network.

Whether for a 3D printer, a camera slider, a custom robot, or a home automation task, this project provides the open-source foundation for powerful and accessible motion control.

The Build

Technical Specifications

The core components that power the project.

  • Core: Raspberry Pi 4 Model B (4GB)
  • Motor: NEMA 17 Stepper Motor
  • Driver: DRV8825 Stepper Motor Driver
  • Software: Python 3, Flask Web Server, RPi.GPIO Library
  • Frame: Custom 3D Printed Mounts & 2020 Aluminum Extrusion
The Logic

Code Snapshot

A quick look at the Python Flask server that powers the web UI.

app_server.py 
from flask import Flask, render_template
import RPi.GPIO as GPIO
import time

app = Flask(__name__)

# Define GPIO pins
DIR_PIN = 20
STEP_PIN = 21

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/move/<direction>/<int:steps>')
def move_motor(direction, steps):
    GPIO.setmode(GPIO.BCM)
    GPIO.setup(DIR_PIN, GPIO.OUT)
    GPIO.setup(STEP_PIN, GPIO.OUT)

    if direction == 'forward':
        GPIO.output(DIR_PIN, GPIO.HIGH)
    else:
        GPIO.output(DIR_PIN, GPIO.LOW)

    for _ in range(steps):
        GPIO.output(STEP_PIN, GPIO.HIGH)
        time.sleep(0.001)
        GPIO.output(STEP_PIN, GPIO.LOW)
        time.sleep(0.001)
    
    GPIO.cleanup()
    return {"status": "success", "steps_moved": steps}

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=80, debug=True)
